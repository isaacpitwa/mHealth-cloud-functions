# mHealth-cloud-functions

> The mHealth-cloud-functions holds the code base for the firefcuntions including cloud APIS with the specific backend logic required for fulll functionality of the application.



## Built With

- Node JS
- Express
- Firebase Firestore
- FIrebase Authentication
- GitFlow
- Javascript

## Development set up
Clone Repository  using 
`git@github.com:isaacpitwa/mHealth-cloud-functions.git` 

OR  using HTTPS

`https://github.com/isaacpitwa/mHealth-cloud-functions.git` 

move into project directory
`cd mHealth-cloud-functions`

Install  Project dependancies using `NPM`

`npm install`

Install  Project dependancies using   `Yarn`

`yarn install`

launch project locally with 

`npm start` or `yarn start`
